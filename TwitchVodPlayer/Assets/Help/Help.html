How it works

  • The user needs to have the video file of the VOD for the player to load and play.
  • You can play the video normally using the player, but if you want to have its chat displayed, you first need to get that VOD's chat file (.cht).
  • Chat logs (.json) can be converted to chat files (.cht) using the Download / Create Chat File tool in the Chat menu.
  • If you don't have it yet, you may also use the same tool to get the VOD's chat log (.json) by using the VOD's ID. The VOD ID is the number at the end of the VOD's URL on Twitch.
  • The created chat file (.cht) needs to have the same filename as the video file and both must be in the same directory, similar to how .srt files work.
  • Then, simply open and load the VOD's video file into the player and it will automatically detect whether there is a chat file for that video.

Loading a VOD

To load a VOD, open its video file.

  • Click Media
  • Click Open VOD
  • Select the video file of the VOD

*VOD Downloaders, such as Twitch Leecher by Franiac, can be used to download the video file of a VOD.

*The VOD's chat filename must be the same as the VOD's video filename, and both must be in the same directory.

Getting the VOD's chat file

To watch a VOD along with its chat, you must first download its chat log (.json) and convert it to a chat file (.cht) Both can be done automatically by the program.

  • Click Chat
  • Click Download / Create Chat File
  • Set the Output Directory for the chat file (.cht)

    • *Make sure it's in the same directory of the VOD's video file for it to be loaded.

    *By default, if a VOD has already been loaded into the player, the Output Directory will be set to the video file's directory.

  • If you already have the chat log (.json), uncheck Use VOD ID to download Chat File and open the file.
  • If you don't have the chat log (.json), check Use VOD ID to download Chat File, then input the ID of the VOD and the program will download it.
  • If your video file doesn't start or end at the same time as the actual VOD on Twitch then check Set begin/end time and input the Begin time and the End time of the video file. This will sync up the chat with the video.

    • *Unchecking Set begin/end time will download the entire chat log of the VOD.

  • Finally, click the Create Chat File button

*If a VOD has already been loaded into the player, the created chat file will automatically have the same filename as the video file, and the program will autoamtically load it

*Otherwise, if a VOD hasn't been loaded yet into the player, the created chat file will have the VOD ID as its filename. Make sure to change this to the filename of the associated video file, and that they are in the same directory.

Downloading Resources (BTTV Emotes, FFZ Emotes, and Channel Badges)

For the chat to use BTTV Emotes, FFZ Emotes, and the channel's badges, you must first download them.

  • Click Resources
  • Click Download Resources
  • Input the name of the channel whose emotes and badges you wish to download.
  • Click the Download Resources button

*The chat will automatically use these resources.

*While the emoticons and badges are still being downloaded, they will be displayed as purple cones. Just wait for a while until the actual emoticon/badge shows up.
